<div align="left">
<h1>tetsuyu</h1>
<i>鉄遊 (tetsu-yū) - A Game Boy emulator, written in Rust.</i>
</div>

<br>

## Features
- Cycle-Accurate CPU
- (Almost) Fully Accurate Audio
- DMG & CGB Support
- Configurable Input
- Configurable Palettes & Shaders
- Cross-Platform
- MBC1/MBC2/MBC3/MBC5 Title Support
- Accurate Color Correction
